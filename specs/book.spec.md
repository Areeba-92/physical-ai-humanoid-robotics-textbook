# Book Specification
## Title
Physical AI & Humanoid Robotics

## Audience
Senior undergraduate and graduate students in AI, Robotics, and Computer Science.

## Learning Goals
- Understand embodied intelligence and Physical AI
- Master ROS 2 for humanoid robot control
- Build digital twins using Gazebo and Unity
- Develop perception and navigation with NVIDIA Isaac
- Integrate Vision-Language-Action (VLA) systems
- Deploy conversational humanoid robots

## Book Structure

### Module 1: Foundations of Physical AI
- What is Physical AI?
- Embodied intelligence vs digital AI
- Humanoid robotics overview
- Sensors in humanoid robots

### Module 2: The Robotic Nervous System (ROS 2)
- ROS 2 architecture
- Nodes, topics, services, actions
- Python agents with rclpy
- URDF for humanoid robots

### Module 3: Digital Twins and Simulation
- Gazebo fundamentals
- Physics, collisions, gravity
- Sensor simulation
- Unity for visualization

### Module 4: NVIDIA Isaac Platform
- Isaac Sim overview
- Synthetic data generation
- Isaac ROS and VSLAM
- Navigation and planning

### Module 5: Vision-Language-Action
- Speech-to-text with Whisper
- LLM-based planning
- Translating language into ROS actions

### Module 6: Humanoid Robot Development
- Kinematics and dynamics
- Bipedal locomotion
- Manipulation and grasping
- Human-robot interaction

### Capstone Project
- Autonomous humanoid robot
- Voice command → plan → navigate → manipulate
